# Frogger Game - Godot C# Learning Project

## Overview

This is a learning project based on the Frogger tutorial from the Udemy course [Learn to Code by Making Games in Godot](https://www.udemy.com/course/learn-to-code-by-making-games-in-godot) by Christian Koch.

## Modifications

This implementation differs from the original tutorial in several ways:

- **Language**: Converted from GDScript to C# for learning purposes
- **Architecture**: Implementing custom architectural patterns and design principles
- **Various enhancements**: Additional changes and improvements made throughout development

## Purpose

This project serves as a hands-on learning experience for:
- Godot Engine with C# scripting
- Game development patterns and architecture
- Practical application of C# in game development

## Credits

Original tutorial and game design by **Christian Koch** via Udemy.

## Technology Stack

- **Engine**: Godot 4.5.1
- **Language**: C# (.NET 8.0)
- **Framework**: Godot.NET.Sdk
